About

Saya tidak pernah secara formal sekolah IT tapi dengan keinginan,
saya terus belajar dari setiap sumber yang bisa saya pakai.

Web site ini saya buat dengan sebatas kemampuan yang saya miliki, saya faham masih banyak kekurangan disana sini, masih banyak hal-hal yang mungkin bisa di eksplor lagi lebih dalam. web ini juga merupakan web awal yang saya kembangkan dengan menggunakan framework bootstrap, banyak percobaan dan eksplorasi untuk membuat web ini. Tapi kemudian masalah-masalah yang muncul saat pembuatan web, itulah yang menarik minat dan perhatian saya, ternyata di sana ada taste yang tidak saya dapat saat mengerjakan hal lain. Semoga sahabat sekalian juga semakin tekun dalam belajar.
